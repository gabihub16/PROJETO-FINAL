# 🎓 Projeto Final 🎓

Este projeto consiste em uma solução completa para estimar o valor de carros usados com base em dados históricos e em um modelo de machine learning. A proposta é ajudar usuários, compradores e vendedores a ter uma referência mais confiável sobre o preço de um veículo antes de realizar uma negociação.

## 🎯 Objetivo 🎯

O objetivo principal do projeto é disponibilizar uma ferramenta prática e acessível para prever o valor de um carro usado a partir de características como marca, modelo, ano, quilometragem, combustível, transmissão e estado geral do veículo. A aplicação reúne uma API para processamento das previsões e um site para interação com o usuário de forma intuitiva.

## ✨ Funcionalidades ✨

- Previsão de preço de carros usados com base em dados reais.
- API REST para receber informações do veículo e retornar uma estimativa.
- Interface web para facilitar o uso da solução sem necessidade de conhecimentos técnicos.
- Experiência simples e rápida para consulta de valores.

## ⚙️ Como funciona ⚙️

1. O usuário informa os dados do veículo na interface ou via API.
2. A aplicação processa essas informações utilizando o modelo treinado.
3. O sistema retorna uma estimativa do valor do carro usado.

## 🌐 Tecnologias utilizadas 🌐

- Python
- Flask ou framework backend utilizado
- Machine Learning para previsão de valores
- Interface web para interação com o usuário

## 🏛️ Estrutura do projeto 🏛️

O projeto é composto por:

- um backend responsável pela API e pela lógica de previsão;
- um modelo treinado para estimar valores;
- uma interface web para facilitar o uso da solução.
